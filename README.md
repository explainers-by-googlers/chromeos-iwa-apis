# ChromeOS APIs for allowlisted Isolated Web Apps

This repository is maintained by the Isolated Web Apps (IWA) team.

It contains the specification and explainers for the set of APIs implemented in
`window.chromeos.isolatedWebApp` that are made available to allowlisted IWAs
running on ChromeOS.
